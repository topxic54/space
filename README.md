# 圆形迷宫 · Circular Maze

一个基于时间驱动的交互式圆形迷宫可视化工具，四个同心圆环按照固定规律旋转，通过时间选择器观察不同时刻的迷宫状态、房间连通关系与访问权限。

---

## 功能特性

- **四圈同心旋转**：外→内四圈分别代表季节、星座、字母房间和祭祀台，各自按照不同方向与频率旋转
- **时间驱动状态**：每个圈的旋转角度由当前时刻精确计算，静止期开门、旋转期关门
- **连通路径检测**：自动检测相邻圈之间对齐的房间，显示可通行路径链
- **昼夜制度**：20:00–07:00 为夜间，仅少数房间（春、秋、狮子、G、祭祀台）保持开放
- **栅栏机制**：固定在 90° 方向，夜间关闭
- **时间播放器**：支持手动拖拽、时间输入、快速跳转及自动播放模式

## 迷宫规则

### 四个圆环

| 圈 | 旋转方向 | 房间数 | 内容 |
|---|---|---|---|
| 第一圈（外） | 顺时针 | 4 | 夏 / 秋 / 冬 / 春（季节） |
| 第二圈 | 逆时针 | 12 | 十二星座（土/水/风/火四元素） |
| 第三圈 | 顺时针 | 6 | M / H / G / C / P / E |
| 第四圈（内） | 逆时针 | 1 | 祭祀台（初始位于 240°） |

### 旋转时序（以整点 n 时为例）

**第一圈**（每小时旋转 180°，2 小时完整一圈）：

| 时段 | 状态 |
|---|---|
| n:00–n:14 | 静止·开门 |
| n:15–n:29 | 旋转 90°·锁门 |
| n:30–n:44 | 静止·开门 |
| n:45–n:59 | 旋转 90°·锁门 |

**第二 / 三 / 四圈**（每小时旋转 90°，4 小时完整一圈）：

| 时段 | 状态 |
|---|---|
| n:00–n:09 | 静止·开门 |
| n:10–n:19 | 旋转 30°·锁门 |
| n:20–n:29 | 静止·开门 |
| n:30–n:39 | 旋转 30°·锁门 |
| n:40–n:49 | 静止·开门 |
| n:50–n:59 | 旋转 30°·锁门 |

### 连通条件

1. 两侧相邻圈**均处于静止状态**
2. 两侧房间**中心角对齐**（容差 ±1°）

### 昼夜制度

- **白天**（07:00–20:00）：所有停止旋转的房间均可通行
- **夜间**（20:00–07:00）：仅 **春、秋、狮子、G、祭祀台** 可通行，其余锁闭

## 核心算法说明

### 旋转角度计算

```typescript
// 以第一圈为例
const hour = Math.floor(t / 60);   // 当前小时数
const w = t % 60;                   // 本小时内已过分钟
const base = hour * 180;            // 累计基础角度

// 本小时内的额外角度
if (w < 15)      extra = 0;               // 静止
else if (w < 30) extra = ((w-15)/15)*90;  // 旋转中
else if (w < 45) extra = 90;              // 静止
else             extra = 90+((w-45)/15)*90; // 旋转中
```

### 连通检测

两个相邻圈房间的全局角度：

```
全局角度 = normAngle(初始角度 + 圈旋转角度)
```

当两圈均静止，且全局角差 < 1° 时，判定为连通。
